# Aplicación para crear tablas de multiplicar

## Instalación
```
npm install
```

## Uso

### Mostrar ayuda
```
node app --help
```
#### Abreviado
```
node app -h
```
## Crear tablas
```
node app --base=8
```
### Alternativa
```
node app --base 5
```
```
node app -b 5
```
## Crear y mostrar en consola
```
node app --base 5 --listar
```
## Crear y mostrar en consola y limitar

Por defecto es hasta 10

```
node app --base 5 --listar --hasta 5
```
## Limpiar tablas
```
node run clear
```
### Con Yarn
```
yarn clear
```
